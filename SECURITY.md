## Reporting a Vulnerability

We take the security of this project very seriously. If you discover a vulnerability, please do NOT report it by creating a public GitHub issue. 

Instead, please report it to the maintainers privately via email or by using GitHub Security Advisories. We will do our best to respond within 48 hours and work with you to plan a fix. 

When reporting, please include:
- A description of the vulnerability.
- Steps to reproduce it (if applicable).
- Any potential impact it might have.
